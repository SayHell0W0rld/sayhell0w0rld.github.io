最近 OpenClaw 在技术圈挺火的，GitHub 星标已经 26 万+，说是能自动操作电脑、管理文件、接入聊天软件。作为一个喜欢折腾工具的人，我决定自己装一下试试。

## OpenClaw 是什么

简单说，OpenClaw 是一个开源的个人 AI 智能体。和 ChatGPT 不同，它不只是聊天，而是真的能帮你干活——操作浏览器、管理文件、发邮件、对接飞书钉钉这些。

关键特点：
- 完全开源，MIT 协议
- 数据存本地，不上传服务器
- 可以接入飞书、钉钉、Telegram 等聊天软件
- 通过 Skills 插件扩展功能

## 安装过程

我用的是 Windows，所以走 PowerShell 路线。

### 第一步：准备环境

OpenClaw 需要 Node.js v22 以上版本。我之前装过，直接检查一下：

```powershell
node --version
```

如果你还没装 Node，去官网下一个就行，https://nodejs.org/zh-cn/download

### 第二步：执行安装脚本

PowerShell 里跑这个命令：

```powershell
iwr -useb https://openclaw.ai/install.ps1 | iex
```

脚本会自动下载 OpenClaw 并启动配置向导。整个过程大概 2-3 分钟，主要看网速。

如果遇到 PowerShell 执行策略问题，先跑一下：

```powershell
Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
```

### 第三步：配置向导

安装完会弹出配置向导，按步骤走：

1. **风险确认**：选 Yes
2. **配置模式**：选 QuickStart（新手推荐）
3. **选择 AI 模型提供商**：这里可以选国外的 Claude、GPT，或者国内的智谱 GLM、DeepSeek 等
4. **填 API Key**：如果用国内模型，去对应平台申请。我用的智谱 GLM-4.7，效果不错价格也便宜
5. **选择聊天平台**：新手建议先跳过，后面可以再配
6. **选择 Skills**：先选 No，后面再装
7. **开启 Hooks**：按空格选中，然后回车

配置完等 30 秒左右，选 Open the Web UI 打开控制面板。

### 第四步：验证

浏览器访问 http://127.0.0.1:18789/ ，能看到界面就说明成功了。在聊天框里输句话试试，能回复就 OK。

## 基本功能体验

装好之后我试了几个功能：

### 文件操作

让 OpenClaw 整理一下桌面文件：

```
帮我整理桌面文件，按类型分类
```

它会自动把图片、文档、安装包归类到不同文件夹。这个功能挺实用的，省得手动整理。

### 浏览器自动化

让它帮我搜个东西：

```
打开浏览器，搜索 OpenClaw 最新版本号
```

它会自动打开浏览器，执行搜索，然后把结果告诉你。不过这里有个问题，它操作的是它自己启动的浏览器，不是你平时用的那个。

### 生成报告

让一个 Excel 文件生成分析报告：

```
读取桌面上的数据.xlsx，生成分析报告
```

它会读取文件，分析数据，生成一个 Word 报告。速度还行，2 分钟左右出结果。

## 接入飞书

很多人用 OpenClaw 是为了对接聊天软件。我试了飞书对接，过程不算复杂：

1. 去飞书开放平台（open.feishu.cn）创建应用
2. 记下 AppID 和 AppSecret
3. 在 OpenClaw 里安装飞书插件：

```powershell
openclaw plugins install @m1heng-clawd/feishu
```

4. 运行 `openclaw config`，选 Channel → Feishu，填入 AppID 和 AppSecret

配好之后在飞书里就能和 OpenClaw 对话了，发消息让它干活。

## 常用命令

装好之后这些命令会经常用到：

- `openclaw gateway status` - 查看运行状态
- `openclaw gateway start` - 启动服务
- `openclaw gateway stop` - 停止服务
- `openclaw doctor` - 自动体检，排查问题
- `openclaw onboard` - 重新跑配置向导
- `openclaw update` - 更新到最新版

## 遇到问题记录

安装过程中遇到几个问题：

1. **npm 报错**：装插件时遇到 `spawn npm ENOENT` 错误，这是 Windows 上的已知 bug。解决方法是修改 OpenClaw 的 node_modules 里的 exec.js 文件，加上 `shell: true` 参数。

2. **命令找不到**：装完之后跑 `openclaw` 命令提示找不到，执行 `exec bash` 刷新一下环境变量就行。

3. **端口冲突**：18789 端口被占用，可以指定其他端口：`openclaw gateway --port 18790`

## 安全注意事项

用 OpenClaw 要注意几点：

- 控制面板默认绑定 127.0.0.1，千万别开放到公网，已经有 900 多个网关被扫描攻击的报告
- API Key 保护好，和密码一样重要
- 装 Skills 插件前看一眼作者和 Star 数，社区里有些插件不太靠谱

## 成本

OpenClaw 软件本身免费。主要成本来自 AI 模型 API 调用。用国内模型的话，智谱 GLM、DeepSeek 这些，一个月几十块钱就够日常使用了。如果用本地模型（需要显卡），成本可以降到零。

## 总结

整体来说，OpenClaw 的安装过程比预想的顺利，大概 10 分钟就能跑起来。功能上确实比单纯的聊天 AI 强不少，能实际操作电脑、管理文件、对接聊天软件。

不过目前还在早期阶段，有些功能不太稳定，插件生态也还在发展中。如果你想折腾一下个人 AI 助手，可以试试；如果追求稳定，可能还要再等等。

对于想快速上手的人，建议：
1. 先用一键脚本安装，别折腾手动配置
2. 模型选国内的，便宜好用
3. 飞书对接可以先搞，比微信靠谱
4. 常用命令记几个，出问题跑 `openclaw doctor` 排查
